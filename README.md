# เครื่องมืออ่านและถอดรหัสข้อมูล Serial (Serial Data Inverter)

โปรเจกต์นี้เป็นสคริปต์ Python (`convert.py`) ที่ออกแบบมาเพื่ออ่านข้อมูลดิบ (Raw Data) จากพอร์ต Serial (เช่น UART) และทำการ "กลับบิต" (Invert Bits) ข้อมูลที่ได้รับ เพื่อให้สามารถอ่านข้อความที่ถูกเข้ารหัสแบบ Inverted UART ได้อย่างถูกต้อง

## ฟีเจอร์หลัก
- **Software Invert**: ทำการกลับค่าสัญญาณ 0 เป็น 1 และ 1 เป็น 0 ผ่านซอฟต์แวร์ เหมาะสำหรับฮาร์ดแวร์ที่ส่งสัญญาณแบบ Inverted แต่ตัวรับไม่รองรับ
- **Real-time Monitoring**: แสดงผลข้อมูลที่ถอดรหัสแล้วทางหน้าจอทันที
- **กรองอักขระที่อ่านไม่ออก**: แสดงผลเฉพาะตัวอักษรที่อ่านได้ (ASCII 32-126) ส่วนอักขระอื่นๆ จะแสดงเป็นจุด (`.`) เพื่อให้อ่านง่ายขึ้น

## สิ่งที่ต้องมี (Prerequisites)
- Python 3.x
- ไลบรารี `pyserial`

## การติดตั้ง

1. ติดตั้ง Python หากยังไม่มี
2. ติดตั้งไลบรารีที่จำเป็นโดยใช้คำสั่ง:

```bash
pip install pyserial
```

## การใช้งาน

1. ตรวจสอบและแก้ไขการตั้งค่าในไฟล์ `convert.py` ให้ตรงกับฮาร์ดแวร์ของคุณ:
   
   ```python
   SERIAL_PORT = 'COM12'   # ระบุพอร์ตที่ใช้งาน (เช่น COM3, /dev/ttyUSB0)
   BAUD_RATE = 113000      # ระบุความเร็ว Baud Rate
   ```

2. เชื่อมต่ออุปกรณ์ของคุณเข้ากับคอมพิวเตอร์
3. รันสคริปต์:

   ```bash
   python convert.py
   ```

4. หากเชื่อมต่อสำเร็จ โปรแกรมจะแสดงข้อความ `[-] Connected to...` และเริ่มแสดงข้อมูลที่ได้รับ

> **ข้อควรระวัง**: ตรวจสอบให้แน่ใจว่าไม่มีโปรแกรมอื่น (เช่น PuTTY, TeraTerm) กำลังใช้งาน Port นั้นอยู่ก่อนที่จะรันสคริปต์
